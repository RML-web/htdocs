Definiciones

HTTP
Protocolo de transferencia de hipertexto (HTTP) es un protocolo de comunicación que se utiliza para la transferencia de información en la World Wide Web. Es fundamental para la comunicación de datos en la web.
HTML
Lenguaje de marcado de hipertexto (HTML) es el lenguaje estándar utilizado para crear páginas web. Define la estructura y el contenido de las páginas web mediante etiquetas y atributos.
URL
Localizador de recursos uniforme (URL) es una secuencia de caracteres que proporciona la dirección única de un recurso en internet. Las URLs se utilizan para localizar recursos como páginas web, imágenes, videos, etc.
TCP/IP
Protocolo de control de transmisión/Protocolo de Internet (TCP/IP) es el conjunto de protocolos de red que permite la comunicación entre dispositivos en internet. Define cómo los datos se transmiten, direccionan, enrutand y reciben en la red.
Internet
Internet es una red mundial de computadoras interconectadas que permite la comunicación y el intercambio de información a través de protocolos estándar. Es una red descentralizada y global.
W3C
Consorcio World Wide Web (W3C) es una organización internacional que desarrolla estándares y protocolos para la web, con el objetivo de garantizar su crecimiento y evolución de manera interoperable.
Mozilla Foundation
Fundación Mozilla es una organización sin fines de lucro que promueve la apertura, la innovación y la participación en internet. Es conocida por su navegador web Firefox y por su compromiso con la privacidad y la seguridad en línea.
